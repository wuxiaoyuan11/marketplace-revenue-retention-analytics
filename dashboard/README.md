# Dashboard Assets

This folder contains the Tableau workbook, a local HTML dashboard, and aggregated CSV files used for business intelligence visualization.

## Tableau Public

Published dashboard:

```text
https://public.tableau.com/app/profile/xiaoyuan.wu/viz/marketplace_revenue_retention_dashboard/MarketplaceAnalyticsOverview
```

## Files

- `marketplace_revenue_retention_dashboard.twb`: Tableau workbook
- `index.html`: local HTML dashboard generated with Python
- `bi_export/`: aggregated dashboard-ready CSV files

The BI export files are intentionally aggregated so the dashboard can be reproduced without committing the raw Kaggle dataset.
